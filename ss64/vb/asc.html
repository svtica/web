Asc

Function that returns the Ascii code which represents a specific character.

Syntax
      Asc (string)

Asc ("A") will return 65

Examples

Dim intDemo

intDemo = Asc ("A")
MsgBox intDemo
> 65

intDemo = Asc ("ABC")
MsgBox intDemo
> 65
