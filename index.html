Outils pour Sysadmins

Ensemble d'outils web moderne avec support des thèmes :

  • 🌙 Mode Sombre - Interface moderne par défaut
  • ☀️ Mode Clair - Interface classique
  • 💾 Mode Retro - Style synthwave nostalgique

Utilisez le sélecteur de thème en haut à droite pour basculer entre les modes !

Liste des outils disponibles

🔧 Générateurs

Générateurs de mots de passe et d'entrées SPF

🌐 Outils réseau

Identification d'adresse IP, calculateur de sous-réseau, identification des classes, conversion d'adresse IP, calculateur CIDR, et superNETs

📧 Analyseur de courriels

Analyse les en-têtes de courriels

🍳 CyberChef

De nombreuses opérations de conversion à travers des recettes. Supporte le chiffrement, encodage, compression et analyse de données

📊 Outils de conversion

Conversion de tailles, de temps de transfert, de dégagement de chaleur et de texte (casse)

💾 Calculateur RAID

Calculateur de RAID, en fonction du nombre de disques et de leur taille. Informations supplémentaires en fonction de chaque type de RAID

🔤 Conversion en Base 64

Encodeur et décodeur de Base64

📝 Analyseur de Log IAS/NPS

Analyse de logs provenant d'IAS (et de NPS - au format IAS)

💿 Simulateur de sauvegardes

Simulateur de sauvegardes Veeam

🔧 Conversions avancées NOUVEAU

Convertisseur multi-unités : température, pression, vitesse, puissance et bases numériques

🔐 Générateur WPA-PSK NOUVEAU

Génération de clés pré-partagées WPA pour diagnostics et configuration réseau

🔒 Calculateur de Hachage NOUVEAU

Calcul MD5, SHA1, SHA256, SHA384, SHA512 pour fichiers et textes - Vérification d'intégrité

🔍 Recherche Commandes SS64 NOUVEAU

Base de données de recherche hors ligne pour commandes Windows, Linux, macOS et PowerShell

🌐 Générateur de Commandes Réseau NOUVEAU

Génère des commandes réseau personnalisées : ping, traceroute, netstat, nslookup, wget/curl, port scan

📅 Calculateur Date/Heure Avancé NOUVEAU

Calculs de dates, conversions de formats, fuseaux horaires, timestamps Unix, jours ouvrés

Ensemble d'outils web pour Sysadmins facilement portables, sans pub, sans cookies, sans tracking, sans prérequis.
Chaque outil est indépendant et peut être utilisé localement sur un poste de travail (sans connexion Internet).
Aucun contenu n'est envoyé à un serveur. Toutes les opérations sont effectuées dans le navigateur web.

OpenSource et hébergé sur GitHub
